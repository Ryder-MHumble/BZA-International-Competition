## Visual Composition Notes

- Banner: oversized serif hero title left/top, subtitle right/bottom, no centered template hero.
- Vision: right-aligned content block, oversized faint numbering, parallax background word.
- Tracks: diagonal text-and-light layout, no equal card grid.
- Registration: sticky progress light column, large underline fields, single-step focus.
- FAQ: sticky monumental title with staggered asymmetric questions.
- Footer: particle return space before information columns.

## Asset Directories

- `public/fonts`: local WOFF2 font files and subsets.
- `public/assets/logos`: academy, partners, social marks.
- `public/assets/images`: event imagery and editorial media.
- `src/webgl/shaders`: particle vertex/fragment shaders.
