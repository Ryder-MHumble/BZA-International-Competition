# Registrations Module

## Public Logic

- Accept registration submissions.
- Validate required fields and shape.
- Record privacy consent and timestamp.
- Persist registration with `submitted` status.
- Return a safe receipt id.

## Admin Logic

- List registrations with pagination and filters.
- Read registration details.
- Update status and notes.
- Export CSV.

## Open Business Rules

- Duplicate email policy.
- Team member structure and limits.
- Minor/guardian consent.
- Advisor requiredness.
